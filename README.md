# Exam Web App

Простое веб-приложение на Node.js + Express, развёрнутое в Yandex Cloud через CI/CD (GitHub Actions).

## Описание

Приложение отображает страницу с сообщением:
> Hello from Yandex Cloud! (deployed via CI/CD)

## Технологии

- **Node.js** + **Express** — веб-сервер
- **Docker** — контейнеризация
- **Yandex Cloud** — инфраструктура (ВМ, Container Registry)
- **GitHub Actions** — CI/CD пайплайныыы
